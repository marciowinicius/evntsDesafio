Comandos utilizados:

express nodewebapp  --> para criar o projeto nodewebapp
use nodewebappdb  --> para criar o banco no mongo
npm install  --> no diretório raiz para instalar as dependências
npm install mongoose --save 
npm install body-parser --save
npm install method-override --save
npm start  --> para testar o aplicativo

Comandos para testar o app:
npm install
npm start
